Monsters Rolodex

The Monsters Rolodex is a web application built using React and TypeScript that allows users to search for and browse through a collection of monsters. The project was designed with a focus on modularization, utilizing a folder structure that keeps each component organized within its own directory. This approach promotes code reuse, making it easier to maintain and scale the application. The use of TypeScript enhances the development experience by providing type safety, allowing for easier debugging and refactoring. The site features a search bar, allowing users to filter through the collection of monsters by name. This project showcases the power of React and TypeScript in creating responsive, efficient, and scalable web applications.

Images Search

The project is a static website for searching and displaying images, created using HTML, CSS, and JavaScript. The website allows users to search for images by entering keywords, which filters and displays the matching images on the page. The website is designed to be visually appealing, with a responsive grid layout and a simple, user-friendly interface. The intention of the project is to provide a basic example of a front-end web application that can be used as a starting point for further development. For example, the search functionality could be expanded to include additional features like pagination, sorting, and filtering options. Additionally, the website could be enhanced with additional pages, content, and functionality, depending on the needs of the project.

Crwn Clothing

This e-commerce project showcases a fully functional online shopping experience using a range of cutting-edge technologies. Built with React, SCSS, and HTML, this app allows users to browse through various departments and purchase their favorite products through an easy-to-use interface. The project also incorporates Firebase platform for authentication and Stripe for secure payment processing. In addition, users can sign up or sign in using their Google accounts. The app's responsive design ensures optimal user experience across all devices. Further development could involve adding more features such as order history and product reviews.
